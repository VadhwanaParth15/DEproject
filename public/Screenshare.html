What is it?

Deskshare is a free, open-source web app that allows you to easily and securely share files directly between devices without uploading them to any server first.It is also used as mirror screening within pc-pc andÂ pc-android.

How to use it?

Sharing files between devices in a local network*

To send a file to another device in the same local network, open this page (i.e. Deskshare) on both devices. Drag and drop a file directly on another person's Device or click the Device and select the file you want to send. The file transfer will start once the recipient accepts the file.
